About Me

Driven by curiosity, powered by determination

My Journey

I'm a self-taught, highly motivated high school student with an intense curiosity for how technology works beneath the surface. From when I was 10, I've been building a hands-on foundation in computer forensics, networking, system configuration, programming, brand design, and creative problem-solving.

I treat my computer like a lab, constantly breaking, fixing, and customizing the devices I use daily. I've developed real skills not from classes, but from figuring things out when everything should've broken.

My approach is simple: Break it. Fix it. Learn from it. This philosophy has led me to start mastering multiple operating systems, configuring complex development environments, and working towards creating solutions that actually work in the real world.

What I'm Capable Of

  • Debugging complex system issues across macOS, Linux, and Windows
  • Configuring multi-environment dev and pen-testing setups
  • Performing penetration testing workflows using Kali Linux
  • Utilizing OSINT methods to reverse engineer and gather forensics information
  • Writing and debugging code in Python, Java, HTML/CSS/JS, Bash, C#, and more
  • Building projects from scratch without hand-holding
  • Creating educational content for YouTube using DaVinci Resolve and Capcut
